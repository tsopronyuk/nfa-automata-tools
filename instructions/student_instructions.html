📚 ALGORITHMS FOR OPERATIONS ON FINITE AUTOMATA

📌 HOW TO FORMAT YOUR ANSWERS IN THE TEST:

  • Q, F - write elements separated by commas, WITHOUT curly braces and WITHOUT spaces
      ✅ q1,q2,q3   ❌ {q1,q2,q3}   ❌ q1, q2, q3
  • Σ - write symbols separated by commas, WITHOUT spaces
      ✅ a,b,c   ❌ {a,b,c}   ❌ a, b, c
  • q0 - write only the state name (e.g., q0 or q1)
  • New transitions - use the format: δ(state,symbol)=set_of_states (without curly braces)
      ✅ δ(q3,a)=q2,q4   ❌ δ(q3,a)={q2,q4}
  • All sets must be sorted alphabetically
      ✅ q1,q2,q3   ❌ q3,q1,q2
  • Only REACHABLE states are included in set Q (unreachable states are removed)
  • The new initial state q0 appears ONLY in the "Union" and "Kleene star (M*)" operations
  • Do NOT write empty sets (just skip them)

1. CONCATENATION (A · B)

Step-by-step procedure:

  1. Take all states from the first automaton A
  2. Take all states from the second automaton B, except its initial state (it becomes unreachable)
  3. Initial state of the resulting automaton = initial state of A
  4. Final states of the resulting automaton:
    • If the initial state of B was final, then final states are all final states of A + all final states of B (except its initial)
    • If the initial state of B was NOT final, then final states are only the final states of B (except its initial)
  5. New transitions: From each final state of A, add the same transitions that were from the initial state of B

✅ EXAMPLE OF A CORRECT ANSWER:

Q = q1,q2,q3,p2,p3
Σ = a,b,c
q0 = q1
F = p3
New transitions: δ(q3,a)=p2

2. UNION (C ∪ D)

Step-by-step procedure:

  1. Create a new initial state named q0
  2. Take all states from C, except its initial state
  3. Take all states from D, except its initial state
  4. Initial state of the resulting automaton = q0
  5. Final states = final states of C (except its initial) + final states of D (except its initial) + q0 (if the initial state of C or D was final)
  6. New transitions: From q0, add all transitions from the initial state of C and all transitions from the initial state of D (combine them)

✅ EXAMPLE OF A CORRECT ANSWER:

Q = q0,r2,r3,s2,s3,s4
Σ = a,b,c
q0 = q0
F = q0,r2,r3,s3
New transitions: δ(q0,a)=r2, δ(q0,b)=s2

3. KLEENE STAR (M*)

Step-by-step procedure:

  1. Create a new initial state named q0
  2. Take all states from M, except its initial state
  3. Initial state of the resulting automaton = q0
  4. Final states = final states of M (except its initial) + q0
  5. New transitions:
    • From q0, add transitions that were from the initial state of M
    • From each final state of M, add transitions that were from the initial state of M

✅ EXAMPLE OF A CORRECT ANSWER:

Q = q0,t2,t3,t4
Σ = a,b,c
q0 = q0
F = q0,t3
New transitions: δ(q0,a)=t2, δ(t3,a)=t2,t4

4. POSITIVE ITERATION (M+)

Step-by-step procedure:

  1. Take all states from M (none are removed)
  2. Initial state of the resulting automaton = initial state of M
  3. Final states = final states of M
  4. New transitions: From each final state of M, add transitions that were from the initial state of M

✅ EXAMPLE OF A CORRECT ANSWER:

Q = t1,t2,t3,t4
Σ = a,b,c
q0 = t1
F = t3
New transitions: δ(t3,a)=t2,t4

⚠️ COMMON MISTAKES

Incorrect Correct Reason
{q1,q2,q3} q1,q2,q3 No curly braces
q1, q2, q3 q1,q2,q3 No spaces
q3,q1,q2 q1,q2,q3 Sorted alphabetically
δ(q3 a)=q2,q4 δ(q3,a)=q2,q4 Comma after state is required
δ(q3,a)={q2,q4} δ(q3,a)=q2,q4 No curly braces around the set

💡 TIP: Before submitting your test, make sure all sets are sorted, there are no spaces, and no curly braces are used!
